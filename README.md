# RAG Project

This is a simple Retrieval-Augmented Generation (RAG) project.

The project is designed to focus on coal-related topics and answer questions by using documents stored in the `articles` folder.

How it works:
- The system retrieves relevant text from the articles folder.
- It uses that retrieved information to generate accurate answers about coal.
- The goal is to keep the responses focused on the coal topic.

Folder structure:
- `articles/`: contains source documents about coal.

This README describes the purpose of the RAG setup and how it uses article documents to produce topic-specific answers.
